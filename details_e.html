Introduction

"Easy Movie Maker" is a powerful video editing tools for users who do not have video post-production experience, provide a more concise operation, do not need to know too many professional terminology, quickly clip the required video. Very suitable for making short video, video courseware, home video, photo MV, YouTube video.

The application provides quick clip mode and project mode based on different user requirements:

Quick Clip Mode provides four of tools for beginning users, Video Trimming, Video Merging, Add Text, Add Music.

Video Trimming:

Clip a video, supports screenshot, capture GIF.

Video Merging:

Merging multiple videos, supports transitions.

Add Text:

Add text to a video.

Add Music:

Add music to a video.

Project Mode for advanced users, supports more media types, provides richer features and options, and is able to save project automatically for the next editing.

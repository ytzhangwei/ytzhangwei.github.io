How do I add text to the video

There are two ways to add text to a video.

1. Click "Add Text" on the main page.

2. Go to the "Text" page while you add or edit a project.

Adjust the start time of the text by moving the slider at the bottom. Double-click the text input box for start text typing, click anywhere outside the text input box to complete the text input. Holding and moving the text input box if you want to change the position of text.

Adjust the start and end time of the text by moving the white button on both sides of the green area at the bottom. Double-click the text input box if you want to modify the text.

Click the green area at the bottom to pop-up the dialog box for the text effects settings.

Click icon go to the text list page.

Click "+" icon to add a new text.
